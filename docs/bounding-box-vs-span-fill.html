Step-by-Step Triangle Rasterization Demo

Both canvases draw the same triangle onto a tiny 80×60 framebuffer, then scale it up. The left side uses a bounding-box edge-function test. The right side uses scanline span filling.

Bounding box + edge functions

This method visits every pixel in the triangle's bounding box. The yellow pixel is the current candidate. Blue pixels are accepted. Dim red pixels were tested and rejected.

Scanline / span fill

This method advances row by row. It computes intersections with the triangle edges, then fills horizontal spans. Orange pixels are accepted. The yellow pixel is the current span pixel.
