Triangle Rasterization Demo

Both canvases draw the same triangle onto a tiny 80×60 framebuffer, then scale it up. The left canvas uses an edge-function inside test. The right canvas uses scanline span filling.

Bounding box + edge functions

This method loops over every pixel in the triangle's bounding box. Each pixel center is tested against the three directed triangle edges.

Scanline / span fill

This method processes the triangle row by row. For each scanline, it computes where the triangle intersects that row, then fills the horizontal span between the intersections.
